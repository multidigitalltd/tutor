יחידות תוכן לימודיות מלאות (וידאו, מצגת, משימות, מבחן סיכום וייעוץ אישי) עם בקרת גישה מבוססת WooCommerce. פותח לפי תקן Multi Digital.

== Description ==

DeSchool מאפשר לבנות "יחידות תוכן" לימודיות גנריות:

* כותרת, תיאור ו"מה כוללת היחידה"
* פרטי מרצה + קישור לקביעת פגישה
* פרקים ללא הגבלה (וידאו מהטמעה/קובץ, מצגת PDF/קישור/הטמעה, תיאור)
* שאלות / משימות לכל פרק עם תיבת תשובה וצירוף קבצים
* מבחן סיכום (בוחן אמריקאי) עם ציון מעבר, הצגת תשובות נכונות וניסיון חוזר
* אזור ייעוץ אישי מסובסד
* שמירת התקדמות, תשובות וקבצים לכל משתמש
* בקרת גישה: צפייה מלאה רק לרוכשי מוצר WooCommerce המשויך ליחידה; preview נעול לאחרים

= עקרונות פיתוח =

* Vanilla JS, ללא jQuery; טעינת נכסים מותנית לעמוד בלבד
* Sanitization מלא בקלט, Escaping מלא בפלט, Nonce והרשאות בכל פעולה
* מבנה נתונים מבוסס Custom Post Types + Meta (ללא טבלאות מותאמות)
* נגישות WCAG 2.2 AA / ת"י 5568, תמיכת RTL/LTR

== Installation ==

1. העלו את התיקייה `md-deschool` אל `wp-content/plugins/`.
2. הפעילו את התוסף דרך תפריט "תוספים".
3. נכנסים לתפריט "DeSchool" ויוצרים יחידת תוכן, או משתמשים ב"יחידת דוגמה" כדי לטעון יחידה מלאה לבדיקה.
4. לכל יחידה ניתן לשייך מוצר WooCommerce שרכישתו פותחת גישה.
5. צרו עמוד "אזור אישי" עם השורטקוד [deschool_account], ועמוד קטלוג (לא חובה) עם [deschool_courses]. ארכיון הקורסים זמין גם בכתובת /unit/.
